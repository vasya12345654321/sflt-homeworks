### Конфигурация HAProxy

Файл: /etc/haproxy/haproxy.cfg
frontend my_frontend
bind *:8080
mode tcp
default_backend my_backend

backend my_backend
mode tcp
balance roundrobin
server s1 127.0.0.1:8001 check
server s2 127.0.0.1:8002 check


---

### Проверка

- http://localhost:8001 → Server 1  
- http://localhost:8002 → Server 2  
- http://localhost:8080 → балансировка через HAProxy  

---

### Важно

Балансировка настроена на уровне L4 (TCP).

В браузере может казаться, что всегда отвечает один сервер —  
это из-за keep-alive (повторного использования соединения).
